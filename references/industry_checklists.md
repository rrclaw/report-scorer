# 行业应有要素清单（Step 0 → 服务维度 5 系统完整性）

> 作用：让"完整"的标准对得上行业。评维度 5 时，选最贴近的一份清单，逐条打钩，**覆盖率 = 打钩数 / 清单条数**，直接映射档位（≥90%→卓越 / ≥75%→强 / ≥60%→合格 / <60%→弱以下）。找不到精确匹配就用通用骨架。清单是"应覆盖"不是"必须长篇"，一句话讲清也算覆盖。

判断"打钩"的标准：该要素被**实质性**讨论（有数据或机制），不是仅提一句名词。

---

## 通用分析骨架（任何行业兜底，11 项）

1. 需求侧：规模、增速、驱动力、结构
2. 供给侧：产能、稼动率、扩产节奏、瓶颈
3. 供需平衡 / 缺口
4. 竞争格局：集中度、份额、梯队
5. 价值链上游：关键投入、卡脖子环节、成本占比
6. 价值链下游：应用、客户、需求传导
7. 价格 / 利差 / 盈利能力趋势
8. 技术路线 / 代际 / 壁垒
9. 周期位置：当前在景气曲线哪一段，历史对照
10. 风险：概率 × 影响，反方情形
11. 估值 / 市场空间 / 催化剂

机读结构化（约占维度 5 的 3 分加分）：核心结论是否结构化、下游可消费。

---

## 化工 / 材料

1. 需求：下游分布、增速、替代/新增需求
2. 供给：在产产能、在建/拟建产能投放时间表、开工率
3. **成本曲线**：行业成本分位、边际成本支撑、谁是边际产能
4. **价差 / 价格**：产品-原料价差、历史分位、盈亏平衡价
5. **库存**：社会库存、企业库存、库存周期位置
6. 原料 / 上游：原料供给、自给率、一体化程度
7. 竞争格局：集中度、龙头成本优势、新进入壁垒
8. 环保 / 安全 / 政策：限产、能耗双控、准入
9. 周期位置 + 历史价差复盘
10. 风险（需求证伪 / 产能超预期 / 原料波动）+ 反方
11. 标的：成本曲线位置、弹性测算、估值

---

## 消费

1. 需求：人群、渗透率、人均消费、增速
2. **品牌力**：心智份额、定价权、提价历史与接受度
3. **渠道**：线上线下结构、渠道库存、动销、铺货
4. **同店 / 复购 / 留存**：单店模型、复购率、会员
5. 竞争格局：市占、新锐冲击、价格带卡位
6. 供应链 / 成本：原料、毛利率趋势、库存周转
7. 单店经济模型 / 门店扩张节奏（如适用）
8. 周期 / 季节性 + 历史复盘
9. 风险（需求降级 / 渠道变革 / 食安舆情）+ 反方
10. 估值：PE/PS、与增速匹配、历史分位
11. 标的分层与催化

---

## 科技 / 制造 / 半导体 / 电子

1. 需求：终端驱动、规模、增速、结构（如 AI/车规/消费分层）
2. 供给：产能、稼动率、扩产周期、良率
3. **技术代际 / 路线**：当前代、下一代、参数对比、壁垒
4. **卡脖子 / 国产化**：海外垄断环节、国产替代进度与纯度
5. 价值链上游：核心材料/设备/IP、卡点、成本占比
6. 价值链下游：客户认证、订单可见度、价值量传导
7. 竞争格局：份额、双寡头/分散、定价权
8. 价格 / ASP / 交期 / 库存
9. 周期位置 + 历史周期复盘（涨幅×股票倍数）
10. 风险（需求证伪 / 产能释放 / 技术路线变更 / 认证不过）+ 反方
11. 标的分层 + 空间/赔率 + 估值重估逻辑

---

## 周期 / 资源 / 能源

1. 需求：终端、增速、季节性、政策驱动
2. **供给约束**：储量、资本开支周期、新增产能滞后、供给刚性来源
3. **成本支撑 / 成本曲线**：90 分位成本、现金成本、边际产能
4. 价格：历史区间、当前分位、期货/现货结构
5. 库存：显性+隐性库存、库存周期
6. 进出口 / 全球供需 / 地缘
7. 竞争格局 / 龙头成本优势 / 集中度
8. 政策 / 环保 / 配额 / 关税
9. 周期位置 + 长周期历史复盘
10. 风险（需求塌方 / 供给超预期 / 政策反转）+ 反方
11. 标的：资源禀赋、成本位置、弹性、估值（PB/重置成本）

---

## 医药 / 生物

1. 需求：患者数、渗透率、支付能力、政策（医保）
2. **临床 / 管线**：在研管线、临床阶段、数据读出、成功率
3. **专利 / 独占期 / 竞争格局**：专利悬崖、me-too 拥挤度
4. 支付端：医保谈判、集采、自费/商保
5. 商业化：销售放量、渠道、学术推广
6. 竞争：同靶点、同适应症、BD/license
7. 监管 / 审批节奏
8. 周期 / 创新周期位置
9. 风险（临床失败 / 集采降价 / 安全性）+ 反方
10. 估值：DCF/管线 NPV/可比、与里程碑匹配
11. 标的分层与催化（数据读出节点）

---

## 金融 / TMT 软件 / 平台

按通用骨架，并补该子行业关键项：
- 金融：息差/资产质量/拨备/资本充足、监管、利率周期
- 软件 SaaS：ARR/NRR/获客成本/留存/Rule of 40、付费意愿
- 平台：GMV/take rate/活跃用户/单位经济模型/网络效应

> 找不到对口清单时，用**通用骨架 11 项**，并在元数据注明"用通用清单评估"。
